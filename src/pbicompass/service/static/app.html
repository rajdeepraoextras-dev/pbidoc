Don't have a .pbip project? Here's how to get one
  1. In Power BI Desktop, go to File → Options and settings → Options → Preview features and enable "Store semantic model using TMDL format" (one-time, if not already on).
  2. Open your report, then File → Save As and choose Power BI project files (*.pbip) as the save type.
  3. This creates a folder named after your report, containing the .pbip file plus a .Report and .SemanticModel subfolder.
  4. Select that whole folder, right-click it, and choose Send to → Compressed (zipped) folder (Windows) or Compress (Mac).
  5. Upload the resulting .zip here.
