FAQ

Before you upload.

Clear answers to the questions Power BI teams ask first.

Does PBICompass read my report data?+

No. The documentation process uses semantic-model metadata such as tables, columns, measures, relationships, DAX and security roles—not the rows inside your report.

What happens to my uploaded file?+

The source is processed inside an isolated workspace and deleted after processing. Generated downloads remain private and expire automatically.

Which Power BI formats are supported?+

The platform supports PBIX and PBIP projects, including modern TMDL models and supported report-layout formats.

Can I use it without AI?+

Yes. A deterministic offline engine can produce complete documentation without using an external AI provider.

Can I regenerate after the model changes?+

Yes. Upload the updated model to create a fresh, consistently structured documentation set.
